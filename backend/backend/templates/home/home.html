Welcome to the Community Event Management Platform

Use command prompt to create , update , delete , view, and RSVP event

This is just for show use cmd to perfom operation

Functionalities are:

    User can signup and login
    Only superuser can create category of event
    All authenticated user can create events using token
    Authenticated user who created event only have a permission to delete or update those events using token
    Any one can see list of events
    Authenticated user can RSVP for events
